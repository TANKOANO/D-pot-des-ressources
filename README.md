# D-pot-des-ressources-de-la-g-n-ration-automatique-de-code-par-hybridation-LLM
Dépot des différentes ressoureces montrant la traçabilité du travail effectué sur le thème de mémoire "Génération Automatique de Code Certifiable pour Systèmes Temps Réel Critiques par Intelligence Artificielle"

- Le dossier **TTool** contient les fichiers de code obtenus par construction, simulation et génération de code dusystème de contrôle de la pression. Langage **UML/MARTE** et **Diplodocus-AVATAR**.
  
- Le dossier **AADL** contient les ressources concernant la modélisation du **pacemaker**

- Le dossier **SCADE** contient les ressources supplémentaires pour la modélisation du système de contrôle de commande du drône.
  
- Le dossier **Simulink** contient les ressources complémentaires pour la modélisation du **système ABS**.

- Le dosier **ressource** contient les codes associés au travail de Master pour le vollet **Étude d'un cas concret**
